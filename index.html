Mendel's Legancy é um sistema de combinação genética baseado na teoria de Mendel.Este sistema oferece a você a oportunidade de explorar as complexidades da hereditariedade e prever as combinações genéticas resultantes de características específicas. Inspirado nas descobertas revolucionárias de Gregor Mendel, considerado o pai da genética moderna, nosso sistema permite que você mergulhe no mundo fascinante da genética e compreenda como as características são transmitidas de geração em geração.

Com nosso sistema, você pode selecionar as características genéticas de interesse, como cor da flor, forma da vagem, altura das flores e muitas outras opções interessantes. Além disso, você pode escolher os organismos que deseja analisar, incluindo seres humanos, plantas, animais domésticos, animais de fazenda e plantas cultivadas. Ao classificar os alelos paterno e materno para cada característica selecionada, você poderá explorar as diferentes combinações genéticas que podem ocorrer.

Com apenas um clique no botão de geração juntamente com as característica selecionas, nosso sistema realizará cálculos precisos e gerará uma tabela completa com os genótipos e fenótipos esperados para cada característica escolhida. Essa tabela fornecerá insights valiosos sobre as possíveis heranças genéticas, permitindo que você compreenda melhor como as características são transmitidas ao longo das gerações.

Não importa se você é um entusiasta da genética ou um estudante curioso, nosso sistema de combinação genética baseado na teoria de Mendel irá cativar sua imaginação e expandir seu conhecimento sobre hereditariedade. Então, embarque nesta jornada emocionante conosco e desvende os segredos ocultos nas combinações genéticas dos organismos selecionados. Prepare-se para mergulhar no mundo intrigante da genética e descobrir as maravilhas da herança genética!

Como o Sistema funciona?

Interface do usuário

O sistema web fornece uma interface do usuário intuitiva onde os usuários podem interagir e selecionar as características que desejam analisar.

Seleção de características

Os usuários poderão escolher quais características desejam analisar, como cor da flor, forma da vagem, altura das flores, entre outros. O sistema fornece uma lista dessas características para que os usuários possam escolher as que desejam.

Combinação de alelos

Com base nas seleções feitas pelos usuários, o sistema deve ter uma base de dados que contenha as informações sobre os alelos correspondentes às características escolhidas. Os alelos podem ser representados por letras, como "A" e "a" para um gene específico que determina uma característica. O sistema deve combinar os alelos do pai e da mãe para prever as possíveis combinações genéticas do descendente.

Geração da tabela

Quando o usuário clica no botão de geração, o sistema realiza os cálculos necessários para combinar os alelos do pai e da mãe e gera uma tabela com as possíveis combinações genéticas. A tabela pode mostrar os genótipos (combinação específica de alelos) e fenótipos (características observáveis) esperados para cada descendente.

Visualização dos resultados

O sistema exibe a tabela gerada para o usuário, mostrando as combinações genéticas previstas com base nas características selecionadas e nos organismos escolhidos. Os usuários podem ver como diferentes características se combinam e quais características são mais prováveis de serem transmitidas aos descendentes.
